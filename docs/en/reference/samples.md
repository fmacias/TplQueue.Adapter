# Samples

Runnable consumer-facing samples are kept in [`TplQueue.Usage`](https://github.com/fmacias/TplQueue.Usage).

- [PackageConsumptionSmokeConsole](https://github.com/fmacias/TplQueue.Usage/tree/main/samples/PackageConsumptionSmokeConsole)
  Package-only smoke consumer for `IJobRoot`, closures, retry, observers, and payload-cache scenarios.
- [QueueObserverConsole](https://github.com/fmacias/TplQueue.Usage/tree/main/samples/QueueObserverConsole)
  Public `Extract -> Transform -> Load` graph with observer subscription and graceful queue completion.
- [QueueObserverSignalRDashboard](https://github.com/fmacias/TplQueue.Usage/tree/main/samples/QueueObserverSignalRDashboard)
  ASP.NET Core sample for named queues, DTO projection, and dashboard-oriented event forwarding.
